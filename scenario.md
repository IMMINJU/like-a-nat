# 편의점 (가제)

## 컨셉
새벽 편의점, 반복되는 밤, 알바생만 점점 인식하게 되는 루프

## 캐릭터

### 알바생 (플레이어)
- 무료함, 관찰자적 시선
- 내면에 뭔가 있지만 본인도 모름
- 루프가 반복되며 점점 기억이 쌓임

### 손님
- 매번 같은 시간에 오는 평범한 사람
- 장난기 있고 엉뚱한 질문을 던짐
- 특별한 존재 아님 - 그래서 더 기묘함

---

## 시나리오

### 1회차 (평범한 밤)

```
[새벽 3시. 형광등 불빛. 편의점 안.]

알바생: (독백) ...3시 12분.

[문 열리는 소리. 손님 입장.]

손님: 안녕하세요.

알바생: 어서오세요.

[손님이 진열대를 둘러본다.]

손님: 삼각김밥 어디있어요?

알바생: 저쪽 냉장고요.

손님: 아, 감사합니다.

[잠시 후. 손님이 삼각김밥과 우유를 들고 온다.]

손님: 이거요.

알바생: 2,400원이요.

[결제.]

손님: 새벽에 일하면 뭐 생각해요?

알바생: ...네?

손님: 손님도 없고, 조용하잖아요. 뭐 생각하나 싶어서.

알바생: 글쎄요... 별 생각 안 해요.

손님: 그게 제일 무서운 거 아니에요?

알바생: ...네?

손님: (웃으며) 농담이에요. 감사합니다.

[손님 퇴장. 문 닫히는 소리.]

알바생: (독백) ...뭐야.

[화면 어두워짐.]
```

### 2회차 (데자뷰)

```
[새벽 3시. 형광등 불빛.]

알바생: (독백) ...3시 12분. ...어?

[문 열리는 소리.]

손님: 안녕하세요.

알바생: (잠깐 멈칫) ...어서오세요.

손님: 삼각김밥 어디있어요?

알바생: (독백) 이거... 아까도...

알바생: 저쪽 냉장고요.

손님: 아, 감사합니다.

[손님이 삼각김밥과 우유를 들고 온다.]

손님: 이거요.

알바생: (독백) 삼각김밥이랑 우유.

알바생: 2,400원이요.

손님: 새벽에 일하면 뭐 생각해요?

알바생: ...이거 아까 물어보지 않았어요?

손님: 네? 저 방금 왔는데요.

알바생: ...아, 아뇨. 죄송해요. 착각했어요.

손님: 피곤하신가 보다. 새벽이니까.

알바생: ...네.

손님: 감사합니다.

[손님 퇴장.]

알바생: (독백) ...뭐지.

[화면 어두워짐.]
```

### 3회차 (확신)

```
[새벽 3시.]

알바생: (독백) 3시 12분. 또야.

[문 열리는 소리.]

손님: 안녕하세요.

알바생: 삼각김밥 저쪽 냉장고요.

손님: ...네? 아, 네. 감사합니다.

[손님, 의아한 표정으로 냉장고로 간다.]

알바생: (독백) 맞았어.

[손님이 돌아온다.]

손님: 이거요.

알바생: 2,400원.

[결제.]

손님: ...저 여기 자주 와요?

알바생: ...네?

손님: 제가 뭘 살지 알고 계신 것 같아서.

알바생: 아... 그냥 감이요.

손님: 감이 좋으시네.

알바생: ...오늘 몇 번째예요?

손님: 네?

알바생: 오늘 여기 오신 거.

손님: 처음인데요?

알바생: ...그렇겠죠.

손님: 이상한 질문 하시네. (웃음) 피곤하신가?

알바생: ...아마도요.

손님: 새벽에 일하면 뭐 생각해요?

알바생: (독백) 또 이 질문.

알바생: ...요즘은 이상한 생각이 들어요.

손님: 어떤?

알바생: 시간이 안 가는 것 같은.

손님: 아, 그거. 새벽이라 그래요. 새벽은 원래 길어요.

알바생: ...그런가요.

손님: 감사합니다.

[손님 퇴장.]

알바생: (독백) 몇 번째지.

[화면 어두워짐.]
```

### 4회차 (대화)

```
[새벽 3시.]

알바생: (독백) ......

[문 열리는 소리.]

손님: 안녕하세요.

알바생: ...안녕하세요.

손님: 삼각김밥 어디있어요?

알바생: 저쪽이요. 참치마요 드실 거죠?

손님: ...어떻게 알았어요?

알바생: 매번 그거 사시니까.

손님: 매번? 저 여기 처음인데.

알바생: ...그쵸.

[손님이 돌아온다.]

손님: 신기하네. 맞췄어요.

알바생: ...우유도요.

손님: (웃음) 뭐야, 무섭게.

알바생: ...저 질문 하나 해도 돼요?

손님: 네.

알바생: 밖에 뭐가 있어요?

손님: 밖이요? 그냥... 어두운 거리?

알바생: 저는 모르겠어요.

손님: 네?

알바생: 마지막으로 밖에 나간 게 언제인지.

손님: ...농담이에요?

알바생: ...아마도요.

손님: (잠깐 침묵) 새벽에 일하면 뭐 생각해요?

알바생: 왜 항상 그걸 물어봐요?

손님: ...네? 처음 물어보는 건데.

알바생: ...그렇죠.

손님: (어색한 웃음) 좀 무서운데요.

알바생: 죄송해요.

손님: 아뇨... 괜찮아요. 감사합니다.

[손님 퇴장.]

알바생: (독백) 밖. 나가본 적 있나.

[화면 어두워짐.]
```

### 5회차 (깨달음)

```
[새벽 3시.]

알바생: (독백) 3시 12분.

[문 열리는 소리.]

손님: 안녕하세요.

알바생: ...

손님: ...안녕하세요?

알바생: 삼각김밥. 냉장고. 참치마요. 우유. 2,400원.
       "새벽에 일하면 뭐 생각해요."

손님: ...뭐예요?

알바생: 당신이 할 말이에요.

손님: (뒤로 한 발) ...저 그냥 갈게요.

알바생: 가도 다시 와요.

손님: ...네?

알바생: 문이 닫히고, 어두워지고, 다시 3시 12분.
       그리고 당신이 들어와요.

손님: ...무슨 소리예요?

알바생: 나도 몰라요. 근데 맞아요.

손님: (침묵)

알바생: ...당신은 기억 안 나죠?

손님: 뭘요?

알바생: 우리 대화.

손님: 처음 만났잖아요.

알바생: ...그쵸.

[긴 침묵.]

손님: ...진짜 괜찮으세요?

알바생: 모르겠어요.

손님: (잠깐 망설임) ...새벽에 일하면 뭐 생각해요?

알바생: (희미하게 웃음) ...끝나면 좋겠다고요.

손님: ...뭐가요?

알바생: 이 밤이요.

[긴 침묵.]

손님: ...저도 그래요. 가끔.

알바생: ...네?

손님: 끝나면 좋겠다고. 새벽이 너무 길어서.

알바생: ...

손님: 감사합니다.

[손님 퇴장.]

알바생: (독백) ...

[화면 어두워지지 않는다. 잠깐의 정적.]

알바생: (독백) ...아침이 오려나.

[희미하게 창 밖이 밝아지기 시작한다. 혹은 다시 어두워진다.]

--- 끝 ---
```

## 결말 분기 (선택)

**A. 아침이 온다**
- 알바생이 무언가를 인정하거나 내려놓음
- 루프 탈출

**B. 다시 반복**
- 화면 어두워지고 "새벽 3시" 다시 시작
- 열린 결말

**C. 플레이어 선택**
- 마지막에 선택지 제공
- "문 밖으로 나간다" / "여기 있는다"

---

## 연출 노트

- 형광등 불빛: 깜빡임으로 불안감
- 시계: 항상 3시 12분 (혹은 미세하게 다름?)
- 손님: 매번 똑같은 동선, 같은 표정
- 알바생: 점점 지쳐가는 모습
